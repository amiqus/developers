The SDK API is still in development and currently provides a limited set of features. There are no pre-compiled SDKs available at this time.

The Amiqus SDK API is used to undertake Amiqus requests on behalf of a data subject.

All SDK API requests require SDK Token authentication. SDK Tokens are generated using the Amiqus API, see Create an SDK token. An SDK Token is scoped to a single record and is valid for 60 minutes.
