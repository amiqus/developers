Endpoints for managing a team's document upload templates ​

List all document upload templates (and List all enabled document upload templates) ​

v1GEThttps://id.amiqus.co/api/team/templates?type=document
v2GEThttps://id.amiqus.co/api/v2/templates/documents

“List all document upload templates” endpoint has been replaced by “Templates → List all document templates” endpoint, which lists all document templates. The url has changed see above.

Request:

Added

  • optional query property enabled (boolean).

Response:

Added

  • object of type paginated_list
  • is_enabled
  • created_at

Changed

  • data.*.object of type document_template (contains an array of objects, so that all properties are nested within this).

Removed

  • status
  • updated_by
  • type

Note: The new filtering option by enabled and the new is_enabled property replaces the List all enabled document upload templates endpoint.
